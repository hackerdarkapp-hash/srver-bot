# بوت الاختراق 🤖

بوت تليجرام متخصص في أدوات الأمن السيبراني (aiogram 3, Python 3.11)

## الأدوات
- 🔎 البحث عن اسم المستخدم
- 🌐 تحليل الموقع
- 📍 تحديد موقع IP
- 📧 فحص البريد الإلكتروني
- 🔐 فحص كلمة المرور
- 🔑 توليد كلمة مرور
- 📷 أدوات QR Code
- 🧠 أدوات التشفير
- 🌍 تحليل الدومين
- 🛡️ كشف VPN/Proxy
- ⚖️ المعلومات القانونية
- 📚 التوعية الأمنية

## التشغيل
```bash
pip install -r bot/requirements.txt
cd bot && python bot.py
```

## متغيرات البيئة
```
TELEGRAM_BOT_TOKEN=...
ADMIN_TELEGRAM_ID=...
```
